# Digital Twin Simulator for IoT Devices

This project simulates a Digital Twin for a single IoT device using Python.

## Features
- Virtual model of device behavior
- Comparison of expected vs actual sensor values
- Anomaly detection and device health status

## Technologies Used
- Python
- CSV (simulated IoT sensor data)

## How to Run
1. Clone or download the repository
2. Run `digital_twin.py`
3. View device health output

## Use Case
- IoT monitoring
- Predictive maintenance
- Digital Twin simulation
